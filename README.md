# Awas Klik — Website Edukasi Anti Penipuan Online

Website statis HTML, CSS, dan JavaScript yang siap di-upload ke GitHub Pages.

## Fitur utama

- Desain modern, bersih, dan ramah low vision.
- Teks besar, kontras tinggi, tombol besar, dan spacing lega.
- Animasi edukatif: hero phone scanning, flow cerita, hotspot tanda bahaya, progress belajar, confetti, quiz, dan simulator pesan.
- Tombol aksesibilitas: perbesar font, mode kontras, kurangi animasi, dan bacakan halaman.
- Tidak memakai framework atau library eksternal.

## Cara upload ke GitHub Pages

1. Upload semua file ke repository GitHub.
2. Buka Settings > Pages.
3. Pilih branch `main` dan folder `/root`.
4. Simpan, lalu buka link GitHub Pages yang muncul.

## Struktur file

```text
phishing_online-main/
├── index.html
├── styles.css
├── script.js
└── README.md
```
